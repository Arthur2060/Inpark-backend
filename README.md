📚 Projeto de Conclusão de Curso
Desenvolvido como requisito para conclusão do curso de Desenvolvimento de Sistemas no SENAI Anchieta, em dezembro de 2025.

O Inpark é um sistema desenvolvido com os seguintes objetivos:

🏢 Gerenciar estacionamentos de forma eficiente

📍 Exibir informações relevantes para motoristas próximos

🚘 Informar disponibilidade de vagas em tempo real

📊 Apresentar dados como nome, localização e demais detalhes do estacionamento

Em resumo a proposta é facilitar tanto a administração dos estacionamentos quanto a experiência dos motoristas que buscam
vagas com rapidez e praticidade.


🚀 Evolução do Projeto

O que começou como um Trabalho de Conclusão de Curso se transformou em algo maior hoje, o inpark é um projeto para aprimorar
nossas habilidades como programadores, parte do nosso portfólio profissional, e um sistema em constante evolução; nosso objetivo com o
sistema atualmente é melhorar funcionalidades já existentes, adicionar novas features, aplicar melhorias práticas baseadas em testes
e aprendizados
